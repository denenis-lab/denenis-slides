Анатомия skill

skill-name/
├── SKILL.md  # обязательный
│   ├── name:  # триггер
│   ├── description:  # когда активировать
│   └── Markdown‑инструкции
├── scripts/  # Python, Bash
├── references/  # документация
└── assets/  # шаблоны, изображения

Только name и description определяют, когда skill активируется. Остальное — по требованию.
